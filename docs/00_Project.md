# 00 Project

## 项目名称

中文名：《像素龙城》

英文代号：Pixel Dragon City

## 一句话简介

一款以龙城大陆为舞台的 2D 像素风动作角色扮演游戏，围绕探索、刷怪、装备成长、职业技能和地下城挑战展开。

## 核心目标

- 第一目标：做出一个可试玩 MVP。
- 第二目标：建立可长期扩展的文档、数据和代码结构。
- 第三目标：形成原创世界观和原创资产生产流程。

## 设计原则

- 操作直接：移动、攻击、拾取、装备都要反馈清楚。
- 成长明确：玩家每次战斗都应接近更高等级、更好装备或新地图。
- 内容数据化：物品、怪物、技能、地图尽量由配置驱动。
- 先小后大：先完成 10 分钟可玩闭环，再扩展职业和世界。

## MVP 范围

MVP 应包含：

- 一个玩家角色
- 一张村庄地图
- 一张野外地图
- 一种普通怪物
- 一种 Boss
- 基础攻击、受击、死亡
- 掉落、背包、装备
- 基础 HUD
- 简单保存与读取

## 非目标

v0.1 到 v0.5 不做：

- 联机
- 大型开放世界
- 复杂经济系统
- 多平台发布
- 商业化付费系统
