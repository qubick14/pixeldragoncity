# 03 World

## 世界名称

龙城大陆

## 世界基调

龙城大陆曾由古代龙族庇护。龙族消失后，人类王国、边境村落、矿洞势力和地下魔物开始争夺遗留的龙晶与古代遗迹。

## 初始区域

### 青木村

新手村，靠近森林和矿道。村民依靠采药、狩猎和矿石运输生活。

主要功能：

- 玩家出生点
- 新手 NPC
- 商人
- 铁匠
- 仓库或存档点

### 黑狼林

青木村外的第一张野外地图。野狼数量异常增加，背后与黑石矿洞的魔气外泄有关。

主要内容：

- 野兔
- 野狼
- 黑狼头目
- 低级草药
- 新手装备掉落

### 黑石矿洞

第一阶段地下城。矿洞深处出现骷髅、矿奴亡魂和晶化怪物。

### 龙城王都

中后期主城。包含职业导师、拍卖、强化、转职和主线剧情。

## 初始故事线

玩家是抵达青木村的无名冒险者。村外怪物异常暴动，村长请求玩家调查黑狼林。击败黑狼头目后，玩家发现黑狼身上嵌有黑色晶片，线索指向废弃的黑石矿洞。
