# 01 Game Design Document

## 游戏类型

2D 像素风 ARPG，单机，俯视角或 2.5D 近似俯视角。

## 核心循环

1. 从安全区域领取目标或整理装备。
2. 进入野外或地下城。
3. 击败怪物，获得经验、金币和装备。
4. 回城补给、出售、强化或更换装备。
5. 解锁更危险地图和更强 Boss。

## 玩家体验关键词

- 爽快刷怪
- 清晰成长
- 像素探索
- 装备惊喜
- 小地图逐步展开

## MVP 玩家流程

1. 玩家出生在青木村。
2. 与村长 `village_chief` 对话，领取第一个任务 `first_hunt`。
3. 进入黑狼林。
4. 击败至少 3 只野狼并拾取掉落。
5. 获得并装备铁剑或皮甲。
6. 击败黑狼头目 `black_wolf_leader`。
7. 回村向村长交付任务，完成第一条闭环。
8. 保存并重新读取后，可以继续或查看已完成的新手流程。

## v0.5 新手流程验收

v0.5 的目标是 10 分钟可玩闭环，不要求一次性完成所有 RPG 系统。验收只看这条路线是否能连续完成：

- 新游戏从 `greenwood_village` 的 `village_spawn` 出生。
- `village_chief` 能开启、推进和完成 `first_hunt`。
- 玩家能在 `greenwood_village` 与 `black_wolf_forest` 之间切换地图。
- 野狼和黑狼头目的击败结果能推动任务状态。
- 掉落、拾取、背包和装备只需要支持铁剑、皮甲、狼皮和药水这些新手流程物品。
- 存档能记录地图、位置、背包、装备和任务进度。

## 核心系统

- 玩家移动
- 战斗与伤害
- 怪物 AI
- 掉落
- 背包
- 装备
- 职业和技能
- 地图切换
- NPC 对话
- 存档

## 内容扩展顺序

优先扩展能增强闭环的内容：

1. 更多怪物
2. 更多装备
3. 更多技能
4. 更多地图
5. 更多任务
6. 更多职业差异
