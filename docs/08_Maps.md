# 08 Maps

## 地图设计原则

- 每张地图有清晰入口、出口和目标区域。
- 新手地图不要太大。
- 地图应支持重复刷怪，但避免玩家迷路。

## 地图列表

### 青木村

功能地图。包含出生点、NPC、商店、铁匠和通往黑狼林的出口。

v0.5 第一版灰盒布局：

- 地图 id：`greenwood_village`
- 出生点：`village_spawn`，位于村庄中心偏南，方便玩家看到村长和出口方向。
- 新手 NPC：`village_chief`，位于出生点北侧或东北侧。
- 暂留功能点：商人、铁匠和仓库只放占位，不实现完整系统。
- 出口：`to_black_wolf_forest`，位于村庄东侧或东南侧。
- 验收重点：玩家不会迷路，30 秒内能从出生点找到村长和黑狼林入口。

### 黑狼林

第一张野外地图。包含野狼刷新点、草药采集点和黑狼头目区域。

v0.5 第一版灰盒布局：

- 地图 id：`black_wolf_forest`
- 入口出生点：`forest_entry`，连接青木村出口。
- 返回出口：`to_greenwood_village`，靠近 `forest_entry`。
- 野狼区域：入口外侧的中部空地，放置多个 `wild_wolf` 刷新点。
- Boss 区域：地图深处，放置 `black_wolf_leader`，用道路、树墙或石块引导玩家前进。
- 草药采集点：可先放占位，不阻塞 v0.5 验收。
- 验收重点：玩家能从入口逐步遇到野狼，再找到黑狼头目，完成后能原路回村。

### 黑石矿洞

第一张地下城。包含狭窄通道、矿车障碍、骷髅和矿洞 Boss。

### 龙城王都

中期主城。未来作为职业、交易和主线剧情中心。

## MVP 测试地图

在正式美术前，先使用灰盒地图：

- 可行走地面
- 不可穿墙障碍
- 玩家出生点
- 怪物刷新点
- 地图边界

## v0.5 地图切换规则

- 地图切换使用稳定地图 id，不使用场景名作为任务或存档语义。
- 青木村到黑狼林：`greenwood_village` / `to_black_wolf_forest` -> `black_wolf_forest` / `forest_entry`。
- 黑狼林回青木村：`black_wolf_forest` / `to_greenwood_village` -> `greenwood_village` / `village_spawn` 或村口返回点。
- 存档记录当前 `map_id` 和玩家坐标；如果读取时坐标无效，则回退到该地图默认出生点。
