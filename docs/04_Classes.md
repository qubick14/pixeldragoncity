# 04 Classes

## 职业设计原则

- 每个职业有明确战斗距离和资源节奏。
- MVP 阶段先实现通用玩家，再逐步拆出职业差异。
- 技能必须有清楚的视觉反馈和冷却反馈。

## 剑士

定位：近战、生存、稳定输出。

核心属性：

- 高生命
- 中攻击
- 高防御

初始技能：

- 普通斩击：近距离攻击。
- 重斩：较慢但伤害更高。
- 格挡：短时间减少受到的伤害。

## 法师

定位：远程、范围、高爆发。

核心属性：

- 低生命
- 高魔法攻击
- 低防御

初始技能：

- 火球术：发射直线飞行法术。
- 冰霜新星：近身范围减速。
- 魔力护盾：消耗魔法抵挡伤害。

## 灵术师

定位：召唤、治疗、持续伤害。

核心属性：

- 中生命
- 中魔法攻击
- 中辅助能力

初始技能：

- 灵符：中距离法术攻击。
- 治愈：恢复生命。
- 召唤灵兽：召唤短时间协助战斗的单位。

## MVP 取舍

v0.1 到 v0.3 先实现剑士式通用近战角色。法师和灵术师在 v0.6 后再进入正式实现。
